# HebronAI v2

HebronAI is an advanced AI chatbot platform built on Next.js, providing powerful conversational AI capabilities with support for multiple LLM providers.

This project is based on [better-chatbot](https://github.com/cgoinglove/better-chatbot) (MIT License) and has been customized and enhanced for the HebronAI platform.
