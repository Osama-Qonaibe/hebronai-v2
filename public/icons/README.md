# PWA Icons

This folder contains the Progressive Web App (PWA) icons for HebronAI.

## Required Icons:

1. **icon-192x192.png** - 192×192 px (Android/Chrome small icon)
2. **icon-512x512.png** - 512×512 px (Android/Chrome large icon)
3. **apple-touch-icon.png** - 180×180 px (iOS/Safari icon)

## Optional Icons:

- **favicon.ico** - 32×32 px (Browser tab icon)
- **favicon-16x16.png** - 16×16 px (Browser tab small)
- **favicon-32x32.png** - 32×32 px (Browser tab)

## Design Guidelines:

- Use HebronAI logo
- Add 10% padding on all sides
- For apple-touch-icon: use non-transparent background
- For Android icons: transparent or colored background is fine
- High quality PNG format
