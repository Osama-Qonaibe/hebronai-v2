## Beginner's Guide - Tools and Concepts Explained

### What are Agents?
**Agents** are custom AI assistants you create for specific tasks.

**Practical Examples**:
- **Customer Support Agent**: Trained to answer questions about your products
- **Content Agent**: Specialized in writing articles and marketing content
- **Programming Agent**: Expert in a specific programming language to help with code

**How to use them?**
1. Go to the "Agents" section in the dashboard
2. Click "Create New Agent"
3. Define specialization and instructions
4. Choose the appropriate model
5. Start conversation with the agent

### What are Workflows?
**Workflows** are automation for repetitive tasks without manual intervention.

**Practical Examples**:
- **Email Summary Workflow**: Automatically reads your email and summarizes important messages
- **Data Analysis Workflow**: Receives Excel file and automatically generates a report
- **Monitoring Workflow**: Tracks a specific site and alerts you on updates

**How to use them?**
1. Go to "Workflows"
2. Choose a ready template or create from scratch
3. Define inputs and outputs
4. Connect steps together
5. Activate the workflow and it will run automatically

### What are MCP Servers?
**MCP (Model Context Protocol)** allows AI to access your tools and private data securely.

**Practical Examples**:
- **MCP for GitHub**: Allows AI to read and write code in your repositories
- **MCP for Database**: Allows querying your data directly
- **MCP for Files**: Allows access to a specific folder on your device

**How to use them?**
1. Go to "MCP Servers"
2. Choose the appropriate server (GitHub, Database, Files...)
3. Enter connection credentials (like API Key)
4. Activate the server
5. Models can now interact with your tools

### What are Smart Tools?

### Tips for Beginners

**Choosing the Right Model**:
- **For general conversations**: GPT-4o or Claude 3.5 Sonnet
- **For programming**: Claude 3.5 Sonnet or DeepSeek-V3

- Monitor your consumption from the dashboard
- Stronger models consume more tokens
- Use lighter models for simple tasks
- Images are counted separately from tokens

**Best Practices**:
- Start with a clear and specific description of what you want
- Use agents for repetitive tasks
- Enable Web Search for current information
- Save important conversations for reference
- Try different models for the same task
